一切皆资源，资源实例化为对象
清单为yaml格式
# 资源类别
- 名称空间级别
- 集群级资源
- 元数据型资源
```bash
kubectl get node -n xxx
# n->namespace 只对名称空间级别资源有效
# 集群级别显示全部
```

# 结构
![](assets/资源清单/file-20260307205138774.png)
```bash
kubectl explain xxx(.xx)
# 查询xxx资源的api字段和说明文档，包括它的group/version,kind等
```
